# QuickStart Guide — myMindMap

Get up and running with myMindMap in minutes.

---

## Installation

### System Requirements
- macOS 13.0 (Ventura) or later
- Mac with Apple Silicon or Intel processor

### Installing the App

1. **Download** myMindMap from the Mac App Store or the official website
2. **Open** the downloaded `.dmg` file
3. **Drag** myMindMap to your Applications folder
4. **Launch** the app from Applications or Spotlight (Cmd+Space, type "myMindMap")

---

## First Launch

When you first open myMindMap, you'll see the Welcome Screen with:

- A prominent **New Map** button
- Recent maps section (empty on first launch)
- Template options for common mind map types

Press **Cmd+N** anywhere in the app to create a new mind map instantly.

---

## Create Your First Mind Map

### Step 1: Start a New Map
- Press **Cmd+N** or click the **New Map** button
- A new canvas appears with a central root node

### Step 2: Name Your Mind Map
- The root node is automatically selected
- Simply start typing to replace "New Idea" with your topic
- Press **Enter** to confirm or **Escape** to cancel

### Step 3: Add Child Nodes
- With the root node selected, press **Tab** to add a child node
- Type your first branch idea
- Press **Enter** to add a sibling node at the same level

### Step 4: Continue Building
- Use **Tab** to add children to any node
- Use **Enter** to add siblings
- Build your structure naturally

### Step 5: Save (Automatic!)
- myMindMap **auto-saves** every change
- No need to manually save—your work is always safe

---

## Basic Operations

### Selecting Nodes
- **Click** on a node to select it
- **Arrow keys** (Up, Down, Left, Right) to navigate between nodes
- The selected node has a blue border

### Editing Node Text
- **Double-click** a node, or
- Select a node and press **Enter**
- Type to edit the text
- Press **Enter** to confirm or **Escape** to cancel

### Deleting Nodes
- Select a node and press **Delete** or **Backspace**
- This removes the node and all its children
- Press **Cmd+Z** to undo

### Moving Nodes
- **Drag and drop** nodes to reorganize
- Or use **Cmd+Up** / **Cmd+Down** to reorder siblings

## Quick Tips

1. **Never lose work**: Auto-save keeps your mind maps safe
2. **Keyboard is faster**: Every feature works without the mouse
3. **Press ?** anytime to see all keyboard shortcuts
4. **Export easily**: Press **Cmd+E** to export as PDF, PNG, Markdown, or JSON
5. **Search**: Press **Cmd+F** to find any node instantly
